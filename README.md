# Crypto Trading Signals

**Crypto Trading Signals** is an automated system for generating and sending trading signals based on technical indicators. It supports multiple cryptocurrency symbols, real-time data from Binance Futures, and Telegram notifications with charts.

---

## Features

- **Real-time OHLCV streaming** using Binance WebSocket API.
- **Technical indicators** for signal generation, including:
  - EMA200 Crossover Signal
- **Automated notifications** via Telegram:
  - Rich messages with Markdown formatting
  - Charts (candlestick + EMA overlay)
  - Support for images and multiple charts
- **Multi-symbol and multi-timeframe support**
- **Configurable thresholds** for each signals
